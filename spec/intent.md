# Intent

## Project Summary

Create a theory-first research project on a canonical continuous-time linear-quadratic differential game. The project should define the players, controls, state dynamics, payoffs, admissibility conditions, and one baseline equilibrium concept, then derive equilibrium conditions and comparative statics in a reproducible way. Unless the project is explicitly re-scoped, the default baseline will use a feedback Nash equilibrium.

## Research Question

How do changes in key parameters of a baseline two-player continuous-time linear-quadratic differential game affect equilibrium strategies, state trajectories, and welfare-relevant outcomes?

## Why This Scope

- Narrow enough for a workshop capstone or first research sprint.
- General enough to support later extensions to asymmetry, stochastic shocks, or alternative equilibrium concepts.
- Compatible with analytic derivation first and optional numeric verification later.

## Deliverables

- Research design memo for the baseline game
- SDD intent, requirements, design, and task specs
- Assumption ledger covering admissibility, existence, and uniqueness conditions
- Comparative-statics roadmap linked to explicit parameters
- Short replication note for any optional verification artifacts

## Non-Goals

- Empirical estimation or calibration to a dataset in the baseline spec
- Multiple equilibrium concepts in the same first-pass analysis
- A large extension family beyond one clearly labeled next-step slot
