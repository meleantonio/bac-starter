# Game Theory Analysis Reference

## Normal-Form Checklist

- Define players and action sets.
- Provide the payoff mapping for every relevant action profile.
- State whether actions are simultaneous, mixed, continuous, or constrained.

## Extensive-Form Checklist

- Define timing, information sets, observability, and chance moves.
- State what is observed at each node.
- If using perfect Bayesian or sequential equilibrium, spell out beliefs and consistency conditions.

## Equilibrium Concepts

- Nash equilibrium: mutual best responses.
- Subgame perfect equilibrium: Nash equilibrium in every proper subgame.
- Perfect Bayesian equilibrium: sequential rationality plus beliefs.
- Mixed strategies: specify support conditions and indifference arguments.

## Comparative Statics

- Name the parameter being changed.
- State which primitives stay fixed.
- Distinguish equilibrium existence from equilibrium selection.

## Common Failure Modes

- Missing information structure or timing.
- Solving a sequential game as if it were simultaneous.
- Ignoring off-path beliefs or credibility.
- Treating multiple equilibria as a single prediction without a selection rule.
